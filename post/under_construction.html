练习（示例题）

我们从一个带中某个位置的 1 开始（请保证你的初始带和带头位置满足这一点，否则示例程序无法运行），将队列中的1变为0，0变为1，最后停止在初始01串的最右端。示例程序做的是先扫描到最左端再返回，并向右直到最右端，进入终止状态。

q0 1 -> q1 2 L
q1 1 -> q1 0 L
q1 0 -> q1 1 L
q1 B -> q2 B R
q2 0 -> q2 0 R
q2 1 -> q2 1 R
q2 2 -> q3 0 R
q3 0 -> q3 1 R
q3 1 -> q3 0 R
q3 B -> e B L
